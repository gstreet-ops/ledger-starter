# GStreet Ledger

Personal small business accounting + tax system for a US, Georgia-based single-member LLC.

## Features (planned)
- Plaid bank sync (Chase, Amex, Truist, Citi, USAA)
- AI-assisted transaction categorization (Claude)
- Double-entry ledger with Schedule C tax mapping
- Quarterly estimated tax calculator
- Year-end CSV/Excel export for tax filing

## Stack
Next.js 15 · Supabase · Drizzle ORM · Plaid · Anthropic API · Tailwind/shadcn · Vercel
